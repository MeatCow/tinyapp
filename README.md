## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- body-parser
- cookie-session


## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the web server using the `node express_server.js` command.
- Access the instance via the `http://localhost:8080/urls` url.
